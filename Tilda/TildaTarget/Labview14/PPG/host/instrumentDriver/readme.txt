This instrument driver allows using a RIO card from National Instruments and
is contained in the NI-FPGA_PPG_Driver.lvlib. 

Please have a look at the description of that library to obtain the following information.

- version number
- author
- maintainer
- entry point for documentation
- license information
- contact
- ...

License information is given also in the textfiles "license.txt" and "GPL.txt".
